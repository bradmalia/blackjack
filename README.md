# Blackjack

A command-line implementation of the classic Blackjack game, written in Python.
<img width="810" height="574" alt="image" src="https://github.com/user-attachments/assets/c5bbf5b1-1e99-4ce1-bc38-0eec3e765fde" />


## Features
- Interactive console GUI.
- Betting system.
- Standard Blackjack rules.
- Shuffled deck with or without jokers.

## Files
- `blackjack.py`: The main game loop and logic.
- `deck_of_cards.py`: Classes for cards and deck management.
- `blackjack_test.py`: Unit tests for blackjack-specific logic.
- `deck_of_cards_test.py`: Unit tests for deck management.
- `player.py`: A player class (currently unused in the main game).

## How to Play
Run the game using Python:
```bash
python3 blackjack.py
```

## Running Tests
To run the tests, use the `unittest` module:
```bash
python3 blackjack_test.py
python3 deck_of_cards_test.py
```
